# Space Invaders
Space Invaders made with Pygame

# Run the game
1. Clone this repo
2. Activate Python virtual env
```py
python -m venv venv/ && source venv/bin/activate
```
3. Install requirement packages
```py
pip install -r requirements.txt
```
4. Run the game
```py
python3 invaders.py
```
